Cocktail Hour is a web application where users can login in to add and save their own drink creations or search a third party API for new beverages.

#### Technologies Used
- React
- Express
- Node Js
- Bootstrap
- HTML
- CSS
- Heroku
- Netlify
- 3rd Party API https://www.thecocktaildb.com/api.php
